 Это изучение основ гита и работы с ветками. Основное направление это приложение с дополнительными функциями работы с данными о банковских операциях пользователя.

В домашнем задании этого урока вы продолжите работать над проектом. Отправите свой локальный Git-репозиторий на GitHub. Начнете работать по GitFlow, разделяя код на ветки, и расширите приложение дополнительными функциями работы с данными о банковских операциях пользователя.
